# online-whiteboard

基于浏览器的实时协作白板系统。

## 项目概述

支持多用户同时在线绘制、标注与沟通，实现低延迟同步与状态一致性，适用于远程会议、教学及团队协作场景。

## 技术栈

- **前端**: React 18, TypeScript, Konva.js, Zustand, Tailwind CSS, shadcn/ui, Vite
- **后端**: Python 3.11+, FastAPI, SQLAlchemy, asyncpg, redis-py
- **数据存储**: PostgreSQL, Redis
- **部署**: Docker, Docker Compose, Nginx
